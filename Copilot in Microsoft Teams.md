# 🧪 Lab — Enhance Collaboration Using Copilot in Microsoft Teams 
## 🎯 Goal
Use **Teams Copilot** to summarize conversations, capture action items, and generate follow-ups efficiently.

---

## 📂 Step 1: Open Microsoft Teams

1. Launch **Microsoft Teams** (Desktop or Web)  
2. Navigate to any **Team, Channel, or Chat**

💡 *Note:* Use a conversation with enough activity for meaningful insights.

---

## 💬 Step 2: Summarize a Conversation

1. Open a **chat or channel thread**  
2. Click **Copilot**  
3. Enter:
```
Summarize this conversation with key points and decisions.
```
💡 *Tip:* If the summary is too long, ask:  

```
Make the summary more concise.
```

---

## 📋 Step 3: Extract Action Items

In Copilot, enter:
```
List all action items from this conversation.
```

✅ Review assigned tasks and next steps  

---

## 🎥 Step 4: Use Copilot in a Meeting

1. Join or open a **recorded meeting** (with transcript enabled)  
2. Click **Copilot** in the meeting window  
⚠️ *Note:* Copilot works best when:
- Meeting transcript is available  
- Recording is enabled  
---

## 📝 Step 5: Generate Meeting Summary

Enter:
```
Summarize this meeting with key discussion points.
```

---

## ✅ Step 6: Capture Decisions and Follow-ups

```
What decisions were made in this meeting?
```

```
List follow-up tasks with owners (if mentioned).
```

---

## 📨 Step 7: Draft a Follow-up Message

```
Write a follow-up message summarizing the meeting and next steps.
```
💡 *Tip:* Adjust tone if needed:
```
Make this message more concise and professional.
```


---

## ✅ Checkpoint

- Chat or meeting summary generated  
- Key **decisions identified**  
- **Action items extracted**  
- Follow-up message created  

---

## 💡 Key Takeaway

**Teams Copilot ensures alignment by summarizing discussions, tracking decisions, and generating follow-ups—saving time and reducing miscommunication.**
