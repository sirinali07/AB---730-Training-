# 🧪 Lab — Copilot for PowerPoint

## Overview
This lab demonstrates how the **same Copilot prompt** produces different outcomes depending on where it is used:
- **Copilot Chat** → Generates a text outline
- **PowerPoint Copilot Agent** → Creates actual slides

---

# 🧪 Lab 1 — Create Content in Chat Mode

## 🎯 Goal
Generate a presentation outline using **Copilot Chat (text only)**.

## 🔧 Steps

1. Open **Copilot Chat** (e.g., in Edge, Bing, or M365 Copilot Chat).
2. Enter the following prompt:
```
Create a 5-slide presentation about AI adoption in retail.
```

3. Review the output:
- Copilot returns a **structured outline**
- Includes slide titles and bullet points
- ❗ No actual PowerPoint file is created

4. Refine the output with a follow-up prompt:
```
Add a slide on ROI and rewrite the titles to be punchier.
```
5. Review the updated result:
- Now contains **5–6 slides**
- Improved, more engaging titles
- Includes ROI-focused content

## ✅ Checkpoint
- You have a **5–6 slide outline in text format**
- Content is structured and presentation-ready (but not a file)

## 💡 Key Learning
- Copilot Chat is best for **planning, brainstorming, and drafting**

---

# 🧪 Lab 2 — Create Content with PowerPoint Copilot

## 🎯 Goal
Generate a **real PowerPoint presentation (.pptx)** using Copilot inside PowerPoint.

## 🔧 Steps

1. Open **Microsoft PowerPoint**
2. Create a **New Blank Presentation**
3. Go to the **Home tab**
4. Click on **Copilot**

5. Enter the same prompt used in Lab 1:
  ```
Create a 5-slide presentation about AI adoption in retail.
```
7. Observe the results:
- Copilot generates **actual slides**
- Includes:
  - Titles
  - Bullet points
  - Layouts and formatting

7. Review the generated slides

## ✅ Checkpoint
- A **real .pptx file with ~5 slides** is created

---

# 🔍 Comparison: Chat vs PowerPoint Copilot

| Feature              | Lab 1 — Chat Mode       | Lab 2 — PowerPoint Agent |
|--------------------|------------------------|--------------------------|
| Output             | Text outline           | Real slides (.pptx)      |
| Acts on files      | ❌ No                  | ✅ Yes                   |
| Best for           | Planning, ideation     | Final deliverable        |
| Interactivity      | Conversational         | Visual + structured      |

---

# 💡 Key Takeaway
> The **same prompt** produces very different results depending on *where* Copilot is used.

- Use **Chat** to think and plan  
- Use **PowerPoint Copilot** to build and deliver
