# 🧪 Lab — Analyze Sales Data Using Copilot in Excel (AB-730)

## 🎯 Goal
Use **Microsoft Excel Copilot** to analyze a real sales dataset, generate insights, create visuals, and perform calculations.

---

## 📥 Step 1: Download the Dataset

1. Open the dataset link in your browser:
```
https://view.officeapps.live.com/op/view.aspx?src=https%3A%2F%2Fraw.githubusercontent.com%2Fsirinali07%2FAB---730-Training-%2Frefs%2Fheads%2Fmain%2FSales.xlsx  
```
2. In the top menu, click **Download**  
3. Save the file as **Sales.xlsx** on your computer  

---

## 📂 Step 2: Open the File in Excel

1. Open **Microsoft Excel**  
2. Click **File → Open → Browse**  
3. Select the downloaded **Sales.xlsx** file  
4. Ensure the dataset is visible  

---

## 🧹 Step 3: Prepare the Data

1. Select any cell inside your dataset  
2. Press **Ctrl + T**  
3. Confirm “My table has headers” → Click **OK**  

✅ This enables Copilot to better understand your data

---

## 🤖 Step 4: Open Copilot

1. Go to the **Home tab**  
2. Click **Copilot**  

---

## 💬 Step 5: Analyze the Data

Enter the following prompts one by one:

### 📊 Generate Insights
```
Analyze this dataset and summarize key trends and insights.
```

### 🏆 Identify Top Performers
```
Which product or region generates the highest revenue?
```

### 📈 Create Visuals
```
Create a chart showing revenue by region.
```

### 💰 Add Calculations
```
Add a column for profit (Revenue - Cost).
```
```
Calculate profit margin percentage.
```

---

## ✅ Step 6: Refine Insights (Optional)

```
What are the key drivers behind high revenue?
```
```
Highlight any unusual trends or anomalies.
```

---

## ✅ Checkpoint

- Data formatted as a **table**  
- Copilot generated **key insights**  
- At least one **chart created**  
- New calculated column (**Profit / Margin**) added  
- Business insights identified  

---

## 💡 Key Takeaway

**Excel Copilot transforms raw data into insights, visualizations, and business decisions quickly—without complex formulas.**
